OPGAVEN - ANIMATION

TEMA 4 gik ud på at udvikle et simpelt klikspil, hvor spilleren interagerer med “gode” og “dårlige” elementer for at optjene point og undgå at miste liv. Spillet er tidsbegrænset, og resultatet afgøres efter få sekunder. Opgaven indebar design og udvikling af grafiske elementer, opbygning af spilsiden med HTML og CSS samt programmering af spillets funktioner og animationer med JavaScript. Der blev også implementeret lyd for at forbedre spiloplevelsen. Spillet skulle tilsidst testet og rettes til.

Fokuspunkter i dette tema

  • Skitsering
  • Illustrator
  • JavaScript
  • Genre & stil
  • Aktivitetsdiagrammer (i vs code)
  • State machine diagram
  • css positionering, events & animation
  • SVG & mp3

Processen

Processen begyndte med idéudvikling og design af spillets visuelle elementer, som først blev tegnet i håndet og derefter tegnet i Illustrator, herunder baggrund, spilelementer og brugergrænseflade. Herefter blev spilsiden opbygget med HTML og CSS, hvor layout og animationer blev implementeret. Spillets funktionalitet blev programmeret med JavaScript, herunder klik-interaktioner, pointoptjening, liv og tidsstyring. Lyd blev tilføjet for at forbedre brugeroplevelsen, og spillet blev løbende testet og justeret baseret på feedback for at sikre både funktionalitet og brugervenlighed.

Løsning

Løsningen blev et funktionelt spil, hvor brugeren kan interagere med “gode” og “dårlige” elementer for at optjene point og undgå at miste liv. Spillet er opbygget med klare visuelle elementer tegnet i Illustrator og stylet med CSS-animationer. JavaScript styrer spillets logik og interaktioner, mens lyd tilføjer ekstra engagement. Resultatet er et responsivt og brugervenligt spil, som fungerer godt både teknisk og visuelt.
